# Bug Bounty & Application Security Write-ups

This repository contains **sanitized, ethical write-ups** of real-world application security findings discovered during bug bounty testing.

## Scope
- Business logic flaws
- Authentication & account recovery issues
- Authorization edge cases
- Real application behavior beyond labs

All targets, endpoints, and identifiers are intentionally removed.

## Purpose
- Demonstrate security testing methodology
- Show clear technical communication
- Share lessons learned from real systems
